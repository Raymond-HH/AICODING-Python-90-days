# 第1周项目：计算器程序

## 项目目标
使用Python基础语法制作一个简单的计算器程序，能够进行四则运算。

## 学习要点
- 变量和数据类型
- 用户输入和输出
- 基本运算符
- 条件语句

## 项目要求
1. 能够接受用户输入两个数字
2. 选择运算符（+、-、*、/）
3. 计算并显示结果
4. 处理除零错误
5. 支持连续计算

## 实现步骤
1. 设计程序流程
2. 实现基本运算功能
3. 添加错误处理
4. 美化输出格式
5. 添加循环功能

## 扩展功能
- 支持更多运算（平方、开方等）
- 添加历史记录功能
- 实现简单的GUI界面
